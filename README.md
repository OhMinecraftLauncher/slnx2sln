# slnx2sln

A small tool which can convert .slnx and .sln files to each other.  
一个用于相互转换 `.slnx` 和 `.sln` 文件的小工具。

## Usage / 用法

```bash
slnx2sln.exe <input_file>
```

The tool automatically detects the input file extension (.slnx or .sln) and performs the corresponding conversion.  
工具会自动检测输入文件的扩展名（.slnx 或 .sln）并执行相应的转换。
